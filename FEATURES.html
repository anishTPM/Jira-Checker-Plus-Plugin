Key Features
Everything your team needs to maintain Jira hygiene

🔍 Smart Validation

Automatically validates every Jira issue on page load. Shows a toolbar button with error count and highlights missing fields with red borders.

➕ Bulk Task Creator

Create multiple Tasks at once from a Story page. Auto-fills Epic, Program, Sprint, Financial Category, and Assignee. All fields mandatory.

🔀 Dual Workflow Support

Switch between Standard (Story → Sub-tasks) and New Workflow (Story → Tasks). Validation rules adapt automatically to your team's process.

📊 Analytics Dashboard

Track scan history, field completion rates, issues fixed, and sync data to Confluence. Export to CSV for reporting.

🚫 Tempo Log Guard

Blocks time logging on issues with pending validations. Shows specific errors inside the Tempo log time popup before allowing hours to be submitted.

🏃 Per-row Sprint Selection

Select a project board in the Bulk Task Creator to load active and future sprints. Each task row has its own sprint dropdown — assign tasks to different sprints independently.

📥 CSV Import / Export

Download a sample CSV template, fill in task details offline, and import back into the Bulk Task Creator. Sprint and Financial Category are matched by name — unmatched values default to blank or self.

💾 Board Preference

The selected sprint board is remembered per project. Next time you open Add Tasks, the board is auto-selected and sprints are loaded automatically.

🔗 Hierarchy Validation

Ensures Tasks and Bugs have Epic Links, and Epics have Parent Links. Keeps your Jira hierarchy clean and traceable.

Workflow Modes
Choose the workflow that matches your team's Jira setup

🟡 Standard Workflow

Story → Sub-tasks (classic Jira)
  • Original Estimate required on Sub-tasks
  • Sub-task 100% logged validation
  • Story must have Sub-tasks (beyond NEW)
  • Story closes when all Sub-tasks done
  • Target dates validated on Sub-tasks

🟢 New Workflow Default

Story → Tasks (linked via "Is a Child of")
  • Original Estimate required on Tasks
  • Task 100% logged validation
  • Story must have linked Tasks (beyond NEW)
  • Story closes when all linked Tasks done
  • Target dates validated on Tasks
  • Bulk Task Creator button on Story pages

➕ Bulk Task Creator
Create multiple Tasks from a Story in seconds — available in New Workflow mode
1

Open any Story page and click "➕ Add Tasks" in the toolbar

2

Modal loads with Story, Epic, Program, and Sprint context auto-filled

3

Fill in Task details — Title, Description, Estimate, Financial Category, Assignee

4

Select a Board to load sprints. Board is remembered for next time

5

Optionally click ⬇️ Sample CSV to download a template, fill it, then 📥 Import CSV

6

Click "+ Add Task" to add more rows. Fields copy from previous row

7

Click "Create All" — Tasks created, linked to Story, Epic set, added to Sprint

Installation
Get started in under a minute
1

Download the latest release from GitHub Releases

2

Open chrome://extensions/ and enable Developer Mode

3

Click "Load unpacked" and select the build/ folder

4

Navigate to any Jira issue — the extension activates automatically
